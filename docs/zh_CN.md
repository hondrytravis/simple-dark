# 为专注而极简

Simple Dark 是一套极简主义 VS Code 主题，包含 **4 个变体**，覆盖暗色和亮色两种使用场景。

## 四款主题

| 主题 | 类型 | 特点 |
|------|------|------|
| **Simple Dark** | 暗色 | 标准平衡的暗色主题，适合日常编码 |
| **Simple Dark Soft** | 暗色 | 低饱和度、柔和对眼睛友好，适合长时间编码 |
| **Simple Light** | 亮色 | 干净高对比度亮色主题 |
| **Simple Light Soft** | 亮色 | 护眼暖纸色背景（参考 Safari 阅读器），适合长时间读写 |

## 设计哲学

- **功能优先**：函数为绿色，参数为橙色——一致的色彩模式帮助你快速解析代码结构
- **语义清晰**：内置原生对象（`DeepSkyBlue`）与自定义类型（`Cyan`）用不同颜色区分，一眼分辨内置类型和自建类型
- **无障碍优先**：全部 4 款主题 **通过 WCAG AA 对比度合规**，暗色正文对比度 8.6:1
- **统一调色板**：4 款主题共享同一套语义颜色映射，暗色/亮色切换时语义一致
